# SocksList
Python Implementation for getting latest Socks Proxy list

## Prerequisites
- Python 2.7
- PhantomJS
- BeautifulSoup
- maxmindb
- Selenium

## Usage
- Run as `python socks_proxy.py`
- Run as `python socks_proxy1.py`
